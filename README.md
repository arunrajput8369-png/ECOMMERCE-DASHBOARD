# ECOMMERCE-DASHBOARD
# Lakshmi Ecommerce Sales Dashboard (Power BI)

##  Project Overview
This Power BI dashboard analyzes the sales performance of Madhav Ecommerce. It provides interactive insights into sales, profit, quantity, customer behavior, payment methods, and product categories to support business decision-making.

---

##  Features
- Interactive Quarterly Filter (Qtr 1 - Qtr 4)
- State-wise Sales Analysis
- Monthly Profit Analysis
- Category-wise Quantity Distribution
- Customer-wise Sales Analysis
- Payment Mode Analysis
- Sub-Category Profit Analysis
- KPI Cards for Sales, Profit, Quantity, and Average Sales

---

##  Dashboard Insights
- Total Sales Amount: **438K**
- Total Profit: **37K**
- Total Quantity Sold: **6K**
- Average Sales: **120K**

### Key Findings
- Maharashtra generated the highest sales.
- Clothing contributed the highest quantity (63%).
- COD was the most preferred payment method.
- Printers generated the highest profit among sub-categories.
- Profit was highest in January and November.
